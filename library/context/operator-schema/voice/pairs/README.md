# Annotated contrastive pairs

The generative core. Each pair shows the distance from generic-AI prose to YOUR voice, with a note on the move. Generate by writing toward the rewrites and applying the moves.

Group files by register (e.g. `builder-pov.md`, `formal.md`, `short-form.md`). Load the ones matching the task; don't load all.

Format per pair:

```
### tag — one-line label
BASE: <generic, highest-probability version>
[YOU]: <same payload, in your voice>
MOVE: <what changed and why — generatively useful, one or two sentences>
register: <which register>
```

Keep ~20–30 total, grouped by register. When a new pair earns in over the cap, replace the weakest/most-redundant — the library swaps, never just grows. One pair per move (dedup). Newest approved work weighted heaviest. A pair earns its place only if it teaches a GENERALIZABLE move — would this note help write a different piece better? If it's a one-off, skip it.

See `example.md` for the format in action.
