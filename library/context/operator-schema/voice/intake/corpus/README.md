# Corpus — raw writing samples

Drop writing here for the agent to extract voice from (run via `voice-setup.md`):
- writing you admire (articles you wish you wrote)
- your own writing (past posts, notes, dictated transcripts, Composio-pulled emails/docs)

Note inside each file whether it's admired or your own. The agent reads these to build `pairs/` and the profile; the raw stays as durable reference.
