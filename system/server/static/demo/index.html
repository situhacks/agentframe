system / server / static / demo livereload

What this is

Stub artifacts that exercise the three preview patterns from runtime/preview-server.md. Real project artifacts live under workspace/projects/<slug>/phase-3-planning/design-language/preview/ and follow the same shapes.

  • Pattern A — direction-compare.html (Design Language sub-artifact 1)
  • Pattern B — design-language.html (Design Language sub-artifact 2)
  • Pattern C — post-1.html (Design Language sub-artifact 3)

How to use

  1. Start the server: python system/server/run.py
  2. Open: http://localhost:8080/system/server/static/demo/index.html
  3. Edit system/server/static/demo/tokens.yaml, then run python -m system.server.lib.tokens_to_css system/server/static/demo/tokens.yaml --out system/server/static/demo/assets/tokens.css — the open tabs auto-refresh.
